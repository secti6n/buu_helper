# buu_helper
A wechat bot

# Requirement

- Python3
- Mysql
- Redis

# Install
0. Install requirements
```
pip install -r requirements
```
1. Setup a mysql database
2. Setup a redis server
3. rename buu_config.py.example to buu_config.py, and write your config to it.
4. running
```
python buu.py
```
